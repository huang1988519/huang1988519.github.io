iOS Gist List

5月 05, 2016

  • MacBook Pro 触摸板和鼠标 飘~~
    现象:
    magic 和 logitech 、 触摸板 均出现 鼠标飘、卡顿的情况。

    重启电脑， 同时按下 command + Option + R + P ，重置 PRAM 和 NVRAM 。 Fixed！

    后来发现是外接显示器转接头的影响，换了一个新的转接头之后再没出现过此现象…囧
