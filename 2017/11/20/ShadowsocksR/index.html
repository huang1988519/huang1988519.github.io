配置VPS

进入VPS界面

Services - My Services - KiwiVM Control Pannel

安装 SSR 服务

在 KiwiVM 界面，选择 KiwiVN Extras - ShadowsocksR Server

然后点击安装。
等待几分钟会提示安装完成。

需要注意，安装完成后，需要重启 VPS，否则就算你在客户端配置正确，依然是无法翻墙。

此处 vps + ssr 安装就完成了，剩下的安装对应平台的客户端即可。
